## Horse Care App

#### Setup:

```
git clone https://github.com/AlexBodnarC/horsecare-app.git
``` 

```
cd horsecare-app
```

Create `.env` file and provide next keys:
**`VITE_SUPABASE_URL`**
**`VITE_SUPABASE_KEY`**

```
npm install
```

```
npm run dev
```

#### Features:

- Authentication
- Registration
- Professional Search
- View All Professionals
- Popular Professionals
- Professional Booking
- Responsive Design
